INTERACTIONS DATASET FILE DESCRIPTION
------------------------------------------------------------------------------------

------------------------------------------------------------------------------------

The file netflix.inter comprising the ratings of users over the movies.
Each record/line in the file has the following fields:  item_id, user_id, rating, timestamp

item_id: the id of the movies and its type is token.

user_id: the id of the users and its type is token. 

rating: the rating of the users over the movies, and its type is float.

timestamp: the UNIX timestamp of the records, and its type is float.
